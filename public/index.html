Everything You Need

A complete AI engineering toolkit — from read-only Q&A to autonomous code changes.

Single Prompt

No mode menus. Describe your goal — the intent router picks Ask, Plan, or Agent automatically.

Voice Input

Press V to record. Whisper, Deepgram, or local whisper.cpp — speech feeds the same pipeline as typing.

Ask Engine

Read-only codebase Q&A. Explain architecture, inspect files, and understand how things work.

Plan Engine

Generate roadmaps and step-by-step execution plans. Select steps and run them sequentially.

Agent Engine

Autonomous file edits, shell commands, and bug fixes with an approval flow before changes land.

Session Memory

Follow-up prompts retain conversation context. Stay in flow without repeating yourself.

GitHub Integration

Review PRs, manage issues, search repos, and create pull requests — all from the terminal.

MCP Support

Connect external tool servers — filesystem, browser, database, Docker — via Model Context Protocol.

Local Models

Run Ollama or LM Studio locally. No cloud API required — full offline AI engineering.

Doctor Command

Health checks for providers, voice, GitHub, MCP, and project context — one command to verify everything.

Project Intelligence

Auto-detects framework, language, package manager, git branch, and status on every session start.

Slash Commands
